Module: Основные команды

Members

<inner> dev

Запускает локальный live-сервер, не минифицирует выходные файлы, пропускает все через валидаторы и форматирует код (eslint, beautyfy js, css, html), чтобы в будущем было его легче поддерживать.

<inner> build

Собирает и минифицирует код для лучшей производительности

<inner> server

Запускает локальный сервер.

<inner> watch

Запускает слежку за файлами и выполнение задач.

<inner> imagemin

Сжимает изображения. Не очень эффективно. Поддерживает файлы: gif,png,jpg,svg,webp. Для лучшего сжатия лучше использовать tinypng.

<inner> tinypng

Сжимает изображения. Очень эффективно. Есть ограничения на 500 файлов в месяц для бесплатного API. Если это число превышенно, то необходимо либо подождать месяц, либо сменить API-KEY для этой задачи, а чуть позже можно снова вернуть старый API-KEY. Поддерживает файлы: png,jpg,webp. Прекрасно подходит для большинство задач.

<inner> convert:to-webp

Конвертирует png,jpg,jpeg,tiff в формат webp.

<inner> convert:from-webp

Конвертирует из webp в png.

<inner> lighthouse

Запускает lighthouse и сохраняет результат проверки в файл.

<inner> pug

Компилятор pug/jade в html. Подключен линтер и БЭМ-валидатор.

<inner> sass

Компилятор sass/scss в css. Подключен автопрефиксер. На выходе имеем обычную версию *.css и минифицированную *.min.css. Sourcemap отключен.

<inner> js

Форматирует и проверяет javascript код. Раставляет правильные отступы, ковычки одного типа ("), раставляет точки с запятой (;). Создает обычную версию *.js и минифицированную *.min.js. Без sourcemap.

<inner> html

Форматирует и проверяет html код. Подключает другие html файлы. Для подробностей см. настройки данного таска.

<inner> gen:doc

Генерирует документацию.

<inner> concat:js

Объединяет несколько *.js файлов в один минифицированный и не минифицированный файл.

<inner> concat:css

Объединяет несколько *.css файлов в один минифицированный и не минифицированный файл.

<inner> clean

Очищает папку с собранным проектом.

<inner> convert:html-to-pug

Конвертирует html в pug.

<inner> convert:encode

Конвертирует кодировку документов.

<inner> convert:to-svg

Конвертирует PNG,JPG,BMP в SVG.

<inner> zip

Архивирует папку с проектом в zip архив.
